To-Do

This is a simple, single page architectured, synchronized to-do list built with AngularJS I built as a learning project.

    I wanted to be able to:
  • see completed or expired items
  • select and view categories
  • have tasks expire after a week
  • use it from my phone or tablet
  • change the theme and color

I used a remote database so it would be accessible anywhere and made it responsive so I could use it on my phone.

I learned a lot making it. It was daunting at the time. Now itâ€™s a reminder that small and simple can be useful and elegant.

    To-Do has:

  • A page that has all the information on every bike in a sortable list. From color to VIN.

  • Two abbreviated lists with only essential data, one for ready bikes, one for offline bikes.

  • A validated form for entering all pertinent data on a new machine.

  • A maintenance log that can be called up by selecting any bike on any list.

  • A quick overview of bike readiness, year and type.

  • Mobile versions of pages that would be useful in the garage
